# Project 2 App Development React Portfolio Website

# Tech Stack

- React 18
- Vite
- React Router
- CSS
- Google Fonts

## Deployment

- This site is deployed using Vercel. Any changes to the main branch automatically updates the deployment. Here is the link: https://portfolio-2lv7.vercel.app/

You can also run the website locally by downloading all the files and using the commands

- npm install
- npm run dev


## Known Limitations

- The contact form does not submit data anywhere, it's just for show until I figure out how
